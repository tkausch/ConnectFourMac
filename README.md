# Connect Four Mac

A macOS command-line tool for Connect Four using Kotlin Multiplatform.

## Features

- Native macOS command-line interface
- Powered by Kotlin Multiplatform via KotlinGameBench.xcframework
- Supports iOS and macOS platforms

## Requirements

- macOS 26.2+
- Xcode 26.3+
- Swift 5.0+

## Building

1. Clone the repository
2. Open `ConnectFourMac.xcodeproj` in Xcode
3. Build and run (⌘R)

## Notes

The project uses library validation exceptions for Debug builds to load the Kotlin framework.
